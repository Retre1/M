Risk Management — 10 Sequential Gates

Every trade must pass ALL gates. Hard blocks force action to 0. Scaling gates reduce position size.

🔴
1. Kill Switch
Emergency halt — manual file (touch data/KILL_SWITCH) or auto-trigger (equity floor 80%, 10 consecutive rejections)
🔴
2. Weekend Gap Guard
Block new trades Fri 20:00 UTC, reduce 50% from Fri 14:00, block all on Sat/Sun
🔴
3. Daily Loss Limit
Hard 2% daily loss limit, auto-resets each calendar day (UTC)
🔴
4. Strategy Filter
Rule-based blackout during high-impact news and market structure breaks
🔴
5. Cooldown
N-bar cooldown after consecutive losses (configurable)
🔴
6. Max Drawdown
5% max drawdown — triggers force-close of all positions
🟡
7. VaR Limit + Portfolio VaR
99% confidence VaR with 252-day lookback. Portfolio VaR uses dynamic correlation matrix
🟡
8. Volatility Targeting
Auto-scale leverage to target 10% annualized portfolio vol. Conservative (0.5x) until 20+ days data
🟡
9. Regime Adaptive
Trending: 1.2x, Mean-Reverting: 1.0x, Flat: 0.7x, Volatile: 0.5x position scaling
🟡
10. Uncertainty (MC Dropout)
Reduces position proportional to model uncertainty. High uncertainty → smaller position

Position Sizing

  • Kelly Criterion (Half-Kelly)
    f* = (win_rate * avg_win - (1-win_rate) * avg_loss) / avg_win × 0.5
  • ATR-Based Stop Loss
    Regime-dependent: Trending 2.5x ATR, Volatile 1.5x, Flat 2.0x, Mean-Reverting 1.8x
  • Concentration Limit
    Max 40% total exposure, 25% per symbol, 0.70 correlation limit

Dynamic Correlations

  • DynamicCorrelationTracker
    Rolling 60-bar window, np.corrcoef, auto-replaces static estimates when 20+ bars available
  • Static Fallback
    EURUSD/GBPUSD: 0.85, EURUSD/USDCHF: -0.90, AUDUSD/NZDUSD: 0.90
  • CorrelationRiskManager
    Portfolio variance w'Cw, Herfindahl concentration, effective exposure check

HiveMind Multi-Agent Ensemble

Three specialist RL agents, each trained on different market conditions, combined via attention-based gating network.

📈

Trend Agent

Specializes in momentum and trend-following. Activated in trending regimes. Wider stops (2.5x ATR), trails profits.

SAC/TQC Momentum
🔄

Reversion Agent

Mean-reversion specialist. Fades extremes, uses Hurst exponent and Bollinger signals. Tighter stops (1.8x ATR), targets mean.

SAC/TQC Mean Reversion
💥

Breakout Agent

Captures structure breaks and volatility expansions. Uses order flow + volume. Dynamic stops, aggressive initial sizing.

SAC/TQC Breakout

Gating Network (TFT Attention)

The gating network dynamically assigns weights to each agent based on current market conditions. Uses Temporal Fusion Transformer attention mechanism to learn which agent performs best in each regime.
